## Domain Terms

| Term | Meaning |
|---|---|
| **REGA** | Saudi Real Estate General Authority (regulator) |
| **GASTAT** | General Authority for Statistics — property classification source |
| **FAL** | Brokerage license required for agencies |
| **Ad license** | Each listing must carry a REGA-issued ad license |
| **RETT** | Real Estate Transaction Tax (5%) |
| **Commission cap** | Max brokerage commission = 2.5% of sale value |
| **Buyer pool** | Shared marketplace where any agent can claim a buyer request |
| **Claim** | An agent reserving a buyer/property request from the pool (expires after 7 days) |
