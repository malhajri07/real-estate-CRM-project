# 00 — Project Overview

## What is Aqarkom?

**عقاركم (Aqarkom)** is a comprehensive Saudi real estate CRM and operating system. It unifies CRM, listings, marketing, deal pipeline, REGA compliance, and tenant management into a single Arabic-first platform.

## Vision

Compete with `aqar.fm` and `PropertyFinder` by offering not only listings but a full **operating system** for Saudi brokers, agencies, and individual agents.

## User Roles

- **WEBSITE_ADMIN** — Platform admins
- **CORP_OWNER** — Agency owners
- **CORP_AGENT** — Agents inside an agency
- **INDIV_AGENT** — Independent agents
- **BUYER / SELLER** — End customers (client portal)

## Core Pillars

1. **CRM** — Leads, customers, activities, contact log
2. **Properties** — REGA/GASTAT classification, 33 types
3. **Pipeline** — Kanban deal stages, forecast
4. **Marketing** — Campaigns, WhatsApp inbox, chatbot
5. **Compliance** — FAL license, ad license, commission cap 2.5%, RETT 5%
6. **Tenant management** — Rent payments, renewals
7. **Buyer pool** — Shared marketplace of buyer requests
